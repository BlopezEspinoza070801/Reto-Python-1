# Reto Python 1

_El proyecto desarrollado trata principalmente del análisis de información de ventas de 4 tiendas para decidir cual cerrar,usando conocimientos básicos de Python_


## Estructura del archivo

1. Se comienza accediendo a las fuentes de datos de las 4 tiendas
2. Se revisan los datos null en caso los tuvieran
3. Se realizan tablas y gráficos básicos para una vista general del estado actual de las tiendas
4. Se sacan conclusiones y se decide que tienda cerrar

## Análisis Realizado

El análisis incluye:

- **Ingresos Totales por Tienda**  
- **Categorías de Productos más y menos vendidas**  
- **Productos más y menos vendidos**  
- **Calificación Promedio de los Clientes por Tienda**  
- **Costo de Envío Promedio por Tienda**  

## Instrucciones de ejecución

_El archivo se desarrollo en Google Colab por lo que los únicos requisitos es descargarlo y abrirlo en la web o en algún otro software como VSCode o Anaconda_

## Tecnologías Utilizadas
* Python
* Pandas (para manipulación de datos)
* Matplotlib (para visualización de datos)
* Google Colab
